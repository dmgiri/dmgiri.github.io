8-Week Roadmap

A structured path from beginner to job-ready developer.


  • filter_1Week 1: Setup & Basics
    Focus: Getting comfortable with the environment.
    • Day 1: Intro & Installation (VS Code)
    • Day 2: Variables & Data Types
    • Day 3: Operators & Calculations
    • Day 4: Type Casting & I/O
    • Day 5: String Manipulation
    • WEEKEND User Info Form Project & Setup Lab
  • filter_2Week 2: Control Flow (Logic)
    Focus: Making decisions and loops.
    • Day 1: Conditional Statements (If-Elif-Else)
    • Day 2: Nested Conditions
    • Day 3: While Loops
    • Day 4: For Loops & Range
    • Day 5: Pattern Printing & Nested Loops
    • WEEKEND Mini Project: Number Guessing Game
  • filter_3Week 3: Data Structures
    Focus: Storing multiple items efficiently.
    • Day 1: Lists Basics & Methods
    • Day 2: Advanced Lists (Sort, Reverse)
    • Day 3: List Comprehensions
    • Day 4: Tuples & Sets
    • Day 5: Algorithms on Lists
    • WEEKEND Project: Student Grade Manager
  • filter_4Week 4: Dictionaries & Functions
    Focus: Key-Value pairs and Modular Code.
    • Day 1: Dictionaries Intro
    • Day 2: Dictionary Methods
    • Day 3: User Defined Functions
    • Day 4: Advanced Functions (*args, **kwargs)
    • Day 5: Modules & Libraries
    • WEEKEND Project: Contact Book (Functions)
  • filter_5Week 5: Files, Errors & OOP
    Focus: Saving data and Object Oriented Programming.
    • Day 1: File Handling (Read/Write)
    • Day 2: Exception Handling (Try-Except)
    • Day 3: Classes & Objects
    • Day 4: Inheritance & Polymorphism
    • Day 5: OOP Real-world Examples
    • WEEKEND Project: Banking System (OOP)
